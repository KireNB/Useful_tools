# Compound Interest Calculator with Two Scenarios

A Python script with a **graphical user interface (GUI)** that calculates and visualizes the growth of two savings scenarios over time.

The chart shows the **annual compound interest growth** for both scenarios and marks any points where the scenarios intersect. The chart can also be downloaded as a PNG file.

---

## Features

- Two independent scenarios with the following inputs:
  - Initial capital
  - Monthly savings
  - Annual interest rate (%)
  - Savings duration (years)

- Annual **compound interest calculation**

- **Interactive, zoomable chart** using Matplotlib

- **Intersection points** of the scenarios are automatically marked

- Option to **download the chart** as a PNG file
